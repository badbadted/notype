🎙️ NoType 安裝與使用手冊

用說的取代打字 — 按住 F9 說話，自動辨識 + AI 潤稿，文字直接出現在游標處。

Windows 10 / 11 v0.1.0 自帶 API Key（BYOK） 常駐系統匣

i這是什麼

NoType 是常駐在 Windows 系統匣的語音輸入小工具。在任何可以打字的地方（Word、瀏覽器、聊天視窗、程式編輯器…），按住 F9 說話、放開停止，它會把你的語音透過雲端 Whisper 轉成文字、用 AI 潤稿（移除贅詞、加標點），再自動貼進你的游標位置。

按住 F9 說話→ 放開→ 語音辨識→ AI 潤稿→ 貼進游標
語音辨識與潤稿走雲端 API，需要網路與一把你自己的 API Key。Key 只存在你本機，不經第三方伺服器。

✓系統需求

  • Windows 10 / 11（64-bit）
  • 麥克風（內建或外接）
  • 網路連線
  • 一把 Groq 或 OpenAI 的 API Key（下方教你拿，Groq 有免費額度）

1安裝

兩種方式擇一：

方式 A — 安裝檔（建議）

  1. 執行 NoType Setup 0.1.0.exe
  2. 一鍵安裝完成，桌面/開始選單會出現 NoType 捷徑
  3. 啟動後圖示常駐在右下角系統匣（藍色麥克風）

方式 B — 免安裝（Portable）

  1. 把 NoType-Portable-0.1.0.exe 放到任意資料夾
  2. 雙擊直接執行，無需安裝
  3. 適合放隨身碟或不想安裝的情境
首次啟動 Windows SmartScreen 可能跳「不明發行者」提示（因為沒做數位簽章）。點「其他資訊 → 仍要執行」即可。

2取得 API Key

服務取得網址說明
Groq（推薦）console.groq.com/keys免費額度大、速度快，金鑰格式 gsk_...
OpenAIplatform.openai.com/api-keys需綁定付款，金鑰格式 sk-...

註冊登入後建立一把新的 API Key，複製起來下一步用。

同一把 Groq Key 同時用於語音辨識與 AI 潤稿，填一次即可。

3首次設定

首次啟動會自動跳出設定視窗（若沒跳，左鍵或右鍵點系統匣圖示 → 設定）。

  1. 辨識服務：選 Groq（預設）或 OpenAI
  2. API Key：貼上金鑰，按「測試」→ 出現綠色「✓ 金鑰有效」表示 OK
  3. 辨識語言：中英混雜建議選「自動偵測」；純中文選「繁體中文」
  4. AI 潤稿：建議開啟，會自動移除「嗯、啊、那個」等贅詞並加標點
  5. 輸入方式：「自動貼上到游標」或「只複製到剪貼簿」
  6. 按「儲存設定」完成

4開始使用

  1. 在任何輸入框點一下，讓游標停在那裡
  2. 按住 F9 不放，螢幕底部出現「錄音中」浮窗
  3. 對著麥克風說話
  4. 放開 F9，浮窗依序顯示「辨識中 → 潤稿中 → 已貼上」
  5. 文字自動出現在你的游標位置 🎉
浮窗狀態：錄音中(紅色波形) → 辨識中 → 潤稿中 → 完成(綠勾) / 錯誤(紅叉)。完成後浮窗自動消失。

⚙設定項說明

設定說明
辨識服務Groq Whisper（large-v3）或 OpenAI Whisper，各自填對應金鑰
辨識語言繁中 / 自動偵測（中英混雜）/ English / 日本語
AI 潤稿開關。風格：口語轉書面 / 保留原語氣 / 精簡贅字 / 翻譯成英文 / 自訂（自己寫 prompt）
輸入方式自動貼上到游標 vs 只複製到剪貼簿
快捷鍵按住 F9 說話（Windows 保留了 Alt+Space，故用 F9）
開機自動啟動開啟後每次開機自動常駐

!疑難排解

症狀處理
金鑰測試顯示「無效」確認金鑰完整複製、沒有多餘空格；Groq/OpenAI 帳號額度是否用盡；網路是否正常
按 F9 沒反應F9 可能被其他程式佔用。本程式會自動退到其他可用鍵——查 debug log 看實際註冊的是哪個鍵
文字沒貼出來確認游標有停在輸入框；某些以系統管理員權限執行的視窗無法被一般程式貼字，試試其他視窗
辨識不準 / 中英混雜怪辨識語言改「自動偵測」；講話清楚、環境安靜；太短的句子（少於幾個字）較不穩
完全沒聲音被辨識檢查 Windows 麥克風權限與預設輸入裝置
程式異常 / 閃退查 debug log（見下方），把內容回報
Debug log 位置： %TEMP%\notype-debug.log
在檔案總管網址列貼上 %TEMP% 按 Enter，找 notype-debug.log 開啟即可看到啟動與錯誤紀錄。

×解除安裝

  • 安裝版：Windows 設定 → 應用程式 → 找 NoType → 解除安裝
  • 免安裝版：直接刪除 exe 檔即可
  • 設定檔位於 %APPDATA%\notype\，要徹底清除可一併刪除該資料夾
